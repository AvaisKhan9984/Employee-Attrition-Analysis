# Employee Attrition Analysis using Python

## Project Overview
This project analyzes employee attrition data to understand the factors affecting employee turnover. The analysis focuses on variables such as age, years at company, and monthly income.

## Objective
To identify patterns in employee attrition and provide insights that can help organizations improve retention.

## Tools and Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
The dataset contains 1470 employee records with 35 columns, including demographic, job-related, and income-related features.

## Steps Performed
1. Loaded the dataset and checked for null values
2. Calculated attrition rate
3. Analyzed Age vs Attrition
4. Analyzed YearsAtCompany vs Attrition
5. Analyzed MonthlyIncome vs Attrition
6. Generated summary statistics and visualizations
7. Drew final conclusions from the analysis

## Key Findings
- Total Employees: 1470
- Employees Who Left: 237
- Employees Who Stayed: 1233
- Attrition Rate: 16.12%
- Employees with attrition were generally younger
- Employees who left had lower tenure
- Employees who left had lower monthly income

## Conclusion
The project shows that age, tenure, and monthly income are important factors associated with employee attrition. This analysis can help HR teams take proactive steps to reduce turnover.
